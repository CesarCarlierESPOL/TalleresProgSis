## Indicaciones generales
* Compilar con *make*
  * Ejecutar **estatico** con *make run-s*
  * Ejecutar **dinamico** con *make run-d*

* Se puede ejecutar el comando **ls -lh /** con *make dir*
* *make all* compilará ambos ejecutables
* Limpieza con *make clean*
